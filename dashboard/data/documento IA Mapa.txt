🔧 Backend (Django)
Modelo de datos

python
class Zona(models.Model):
    nombre = models.CharField(max_length=100)
    color = models.CharField(max_length=7, default="#FF0000")  # hex color

class CodigoPostal(models.Model):
    cp = models.CharField(max_length=5)
    colonia = models.CharField(max_length=100)
    zona = models.ForeignKey(Zona, on_delete=models.CASCADE, related_name="codigos_postales")
Generación de GeoJSON

Puedes crear un endpoint REST que devuelva las zonas en formato GeoJSON.

Ejemplo con Django REST Framework:

python
from rest_framework.views import APIView
from rest_framework.response import Response

class ZonasGeoJSON(APIView):
    def get(self, request):
        zonas = Zona.objects.all()
        features = []
        for zona in zonas:
            # Aquí deberías tener guardados los polígonos (coordenadas) de cada zona
            features.append({
                "type": "Feature",
                "properties": {
                    "nombre": zona.nombre,
                    "color": zona.color,
                },
                "geometry": {
                    "type": "Polygon",
                    "coordinates": zona.coordenadas  # lista de lat/lon
                }
            })
        return Response({"type": "FeatureCollection", "features": features})
Las coordenadas pueden venir de un archivo base (shapefile, GeoJSON inicial) o calcularse con librerías como Shapely.

L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {

fetch('/api/zonas-geojson/')
    .then(res => res.json())
    .then(data => {
        L.geoJSON(data, {
            style: function(feature) {
                return { color: feature.properties.color };
            },
            onEachFeature: function(feature, layer) {
                layer.bindPopup("Zona: " + feature.properties.nombre);
            }
        }).addTo(map);
    });
